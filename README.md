** Greenfoot İlk Oyun **

Bu proje, Greenfoot kullanılarak geliştirilmiş 2D Java tabanlı bir oyun projesidir.  
Amaç, Java programlama mantığını ve nesne yönelimli programlamayı (OOP) pekiştirmektir.

---

 Proje Hakkında

Oyunda oyuncu karakteri klavye ile kontrol edilir.  
Amaç, ekrandaki nesneleri toplayarak skor kazanmaktır.

Proje eğitim amaçlı olarak geliştirilmiştir.

---

 Kullanılan Teknolojiler

- Java
- Greenfoot
- Object Oriented Programming (OOP)

---

 Oyun Kontrolleri

-  Sol ok: sola hareket  
- Sağ ok: sağa hareket  
-  Yukarı ok: ileri hareket  



---

 Nasıl Çalıştırılır?

1. Greenfoot programını açın  
2. Proje klasörünü seçin  
3. `Run` butonuna basın
